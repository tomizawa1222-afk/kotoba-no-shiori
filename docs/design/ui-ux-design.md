# UI/UX デザイン設計書

## 1. デザインコンセプト

**「Minimal & Energetic」**

読書という静的な行為と、心動かされる瞬間の動的な感情を対比させたデザインシステムを採用します。

- **Minimal (静)**: コンテンツ（言葉）を主役にするため、ベースは徹底的にノイズを排除したモノトーン。
- **Energetic (動)**: 「刺さる」「閃く」といった感情の動きや、ユーザーのアクション（投稿、いいね）を、鮮やかなイエロー〜オレンジのグラデーションで表現。

### デザイン原則
1. **Content First**: UIパーツよりも、ユーザーが投稿した「言葉」が最も目立つようにタイポグラフィを調整する。
2. **Optimistic Feedback**: 「いいね」や「投稿」のアクションに対し、即座にフィードバック（アニメーション等）を返し、心地よい操作感を提供する。
3. **Thumb-Friendly**: モバイルでの利用を前提とし、主要な操作ボタンを画面下部に集約する。

## 2. デザインシステム

### 2.1 カラーパレット

| 色名 | カラーコード | 用途 |
| :--- | :--- | :--- |
| **Primary Black** | `#000000` | メインテキスト、見出し、アイコン（通常時） |
| **Background White** | `#FFFFFF` | アプリ背景、カード背景 |
| **Surface Gray** | `#F5F5F5` | 入力フィールド背景、ボーダーライン |
| **Text Secondary** | `#666666` | 補足情報、タイムスタンプ |
| **Accent Gradient** | `linear-gradient(135deg, #FFFC00 0%, #FFA500 100%)` | プライマリーボタン、アクティブタブ、いいね（Active） |
| **Error Red** | `#EF4444` | エラーメッセージ、削除ボタン |

### 2.2 タイポグラフィ

システムフォント (`-apple-system`, `BlinkMacSystemFont`, `Roboto`, `Noto Sans JP`) を使用し、OSネイティブな読みやすさを提供します。

| スタイル名 | サイズ / Weight | 行間 (Line Height) | 用途 |
| :--- | :--- | :--- | :--- |
| **Display** | 24px / Bold | 1.4 | 画面タイトル |
| **Heading** | 18px / Bold | 1.4 | セクション見出し、ユーザー名 |
| **Body (Quote)** | 16px / Regular | **1.8** | 引用本文（ゆったりとした行間で可読性を確保） |
| **Body (Default)** | 14px / Regular | 1.5 | 一般的なテキスト |
| **Caption** | 12px / Regular | 1.4 | タイムスタンプ、メタ情報 |

## 5. インタラクション設計

- **ローディング**: 画面全体をブロックするローディングは極力避け、スケルトンスクリーン (Skeleton Screen) を使用して体感待ち時間を短縮する。
- **無限スクロール**: タイムラインの最下部に到達する前に次のデータをプリフェッチする。
- **マイクロインタラクション**:
  - 「いいね」ボタン押下時に、ハートが弾けるようなパーティクルアニメーションを表示。
  - タブ切り替え時に、コンテンツが左右にスライドするようなトランジションを入れる（任意）。
